# Sistema de Pesquisa de Livros

Este projeto é um sistema de pesquisa de livros onde você pode adicionar e visualizar livros. O backend é construído com Node.js e o frontend utiliza EJS e Bootstrap.

## Tecnologias Utilizadas

- **Backend**: Node.js, Express
- **Frontend**: EJS, HTML, Bootstrap, JavaScript
- **Gerenciamento de Dependências**: npm

## Como Executar o Projeto

1. Navegue até o diretório do backend:
   ```bash
   cd backend
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor:
   ```bash
   node server.js
   ```

4. Acesse o sistema de pesquisa de livros pelo navegador em `http://localhost:3007`.

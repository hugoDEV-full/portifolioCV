# Blog Pessoal

## Descrição
Um blog simples onde os usuários podem criar, editar e excluir postagens. Também permite a adição de comentários em tempo real com WebSocket.

## Tecnologias Utilizadas
- JavaScript
- EJS
- Bootstrap
- Node.js
- WebSocket

## Funcionalidades
- Criar postagens de blog
- Editar postagens existentes
- Excluir postagens
- Adicionar e visualizar comentários em tempo real

## Como Executar
1. Clone o repositório.
2. Instale as dependências: `npm install`
3. Execute o servidor: `npm start`
4. Acesse `http://localhost:3000` no seu navegador.
